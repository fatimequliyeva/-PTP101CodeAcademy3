# QA və Code Review Hesabatı

## Funksional yoxlama

- Contact səhifəsinin layout-u screenshot-a uyğunlaşdırıldı
- Form validation mesajları düzgün göstərilir
- Aksiyon düyməsi aktiv və funksionaldır

## Accessibility və semantika

- Input/textarea elementləri üçün aria-label və aria-invalid əlavə olundu
- Xəta mesajları üçün role="alert" istifadə edildi
- Semantik struktur qorunub saxlanıldı

## Cross-browser

- Lokal mühitdə Chrome əsaslı yoxlama aparıldı
- Edge/Safari/Firefox üzrə real cihazlarda test edilməyib

## W3C uyğunluğu

- HTML atributları valid formata uyğunlaşdırıldı
- Form elementləri üçün label/aria əlaqələri yaradıldı

## Performance

- Əlavə ağır assetlər əlavə edilmədi
- Lighthouse ölçümləri bu mühitdə icra edilmədi
